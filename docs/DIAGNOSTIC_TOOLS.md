# Diagnostic Tools

Tools for debugging mesh data and frame extraction issues.

**Available Tools:**
- `query-mesh-db.js` - Query MongoDB mesh data
- `backend/debug-mesh-cache.js` - Debug mesh cache
- `backend/diagnose-frame-issue.js` - Diagnose frame extraction issues

**Usage:**
```bash
node query-mesh-db.js
node backend/debug-mesh-cache.js
node backend/diagnose-frame-issue.js
```

See individual files for detailed usage.
